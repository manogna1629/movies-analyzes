# 🎬 Movies Analysis Project

This project focuses on analyzing movie scripts using Python. It extracts meaningful insights such as character appearances, dialogues, emotions, and interactions from movie scripts and visualizes them using different techniques.

---

## 📌 Project Overview

The goal of this project is to perform **data analysis on movie scripts** and generate insights such as:

- Character appearances
- Dialogue distribution
- Character interactions
- Sentiment & emotion analysis
- Scene-level information
- Gender distribution

---

## 🚀 Features

- 📊 Character appearance analysis
- 🗣️ Dialogue frequency tracking
- 🔗 Character interaction mapping
- 😊 Sentiment & emotion analysis
- 🎥 Scene segmentation (location, time, type)
- ⚖️ Gender distribution analysis
- 📈 Data visualization using plots & graphs

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Regular Expressions (Regex)
- NLTK (Natural Language Toolkit)
- Plotly
- Seaborn
- NetworkX
- Jupyter Notebook

---

---

## ⚙️ How It Works

1. **Data Collection**
   - Movie scripts are collected and processed.

2. **Data Cleaning & Segmentation**
   - Scripts are divided into scenes using Regex.

3. **Feature Extraction**
   - Characters, dialogues, and scene data are extracted.

4. **Analysis**
   - Character frequency
   - Interactions
   - Emotional tone

5. **Visualization**
   - Graphs and plots are generated for insights.

---

## 📊 Example Outputs

- Character appearance graphs
- Interaction networks
- Emotion trends across the movie
- Dialogue distribution plots
